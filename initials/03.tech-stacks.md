# Technical Stack: Decidely.ai

This document outlines the core technologies, frameworks, and deployment services used to build Decidely.ai. The stack is optimized for rapid development, multi-agent AI orchestration, and staying within the $5/month hackathon budget.

## 1. Frontend (User Interface)
- **Framework:** React
- **Build Tool:** Vite (for fast HMR and optimized builds)
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios (or standard Fetch API)
- **Package Manager:** Bun (for faster dependency resolution and script execution)

## 2. Backend API & Agent Orchestration
- **Language:** Python 3.10+
- **API Framework:** FastAPI (for high-performance, async RESTful endpoints)
- **Server:** Uvicorn (ASGI web server)
- **Data Validation:** Pydantic
- **AI Framework:** Google Agent Development Kit (ADK) - `google-adk` (for defining the Supervisor routing pattern and sub-agents)
- **Package Manager:** uv (extremely fast Python package installer and resolver)

## 3. AI & Models
- **LLM Provider:** Google Vertex AI (or Google AI Studio)
- **Primary Models:** Gemini 3.1 Flash Lite Preview
- **Grounding Tool:** Google Search Grounding (built-in Gemini tool for real-time web data)

## 4. Model Context Protocol (MCP) Integrations
- **Core Data Structuring:** SQLite MCP (Used by the Evaluator Agent to build, query, and score the decision matrix)
- **Knowledge Export (Stretch Goal):** Google Drive MCP (Used by the Primary Agent to save final reports to the user's workspace)

## 5. Database & Context Memory
- **Service:** Google Cloud Firestore (Native Mode)
- **Purpose:** Storing chat session histories, user preferences, and structured decision matrix outputs (JSON).
- **Benefit:** Generous free tier perfectly fits the budget constraints.

## 6. Infrastructure & Deployment
- **Backend Hosting:** Google Cloud Run (Containerized via Docker, scales to zero to save costs)
- **Frontend Hosting:** GitHub Pages (Free tier)
- **Containerization:** Docker (for the FastAPI backend)

## 7. Development Environment
- **Shell:** Fish Shell (activation scripts customized for `source .venv/bin/activate.fish`)
- **Version Control:** Git / GitHub
